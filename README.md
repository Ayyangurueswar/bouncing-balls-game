# bouncing-balls-game
Bouncing balls game created using JavaScript.
## Technologies used:
1. JavaScript
2. HTML
3. CSS
## How to play?:
Move the player ball toward the balls moving randomly and try to capture as many balls as possible.
### Controls:
1. W - Move upwards
2. D - Move rightwards
3. S - Move downwards
4. A - Move leftwards.
